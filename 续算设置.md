## 故障排除

### 如果续训失败

**错误1: 模型架构不匹配**
```
Error: size mismatch for encoder.*: expected ... but got ...
```
**解决**: 检查模型配置是否与中断时一致

**错误2: 检查点文件缺失**
```
Error: No such file or directory: 'ckpt-0-tr/model.safetensors'
```
**解决**: 确认检查点文件路径正确

**错误3: 优化器状态加载失败**
```
Error: Could not load optimizer state
```
**解决**: 可以忽略优化器状态，只加载模型权重继续训练
